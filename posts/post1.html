How to Start Learning Python — Free Resources

April 19, 2024

Step 1: Start with the basics

Go to Python.org and download Python. Then start with these free resources:

  • Automate the Boring Stuff (free online book)
  • freeCodeCamp's Python course on YouTube (4 hours)
  • W3Schools Python tutorial (quick reference)

Step 2: Practice daily

Spend 30 minutes every day coding. Use:

  • HackerRank — Python track
  • LeetCode — Easy problems
  • Your own small projects (calculator, to-do list, etc.)

Step 3: Build something real

Once you know the basics, build a project. Anything. A weather app, a data visualizer, a web scraper.

Resources I personally used:

  • freeCodeCamp Python (4 hours)
  • Automate the Boring Stuff
  • W3Schools Python
